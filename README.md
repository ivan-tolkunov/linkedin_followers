# LinkedIn followers

This is a service for gaining LinkedIn followers.

## How to use

Clone repository:

```bash
git clone git@github.com:ivan-tolkunov/linkedin_followers.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Then run the server:

```python
python app.py
```
## Test

http://127.0.0.1:5000/get_follower_count?linkedin_url=https://www.linkedin.com/in/ivantolkunov/
